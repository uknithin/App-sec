Top 3 High-Impact Vulnerability Mapping

1. Account Takeover

Credential Stuffing / Brute Force

Attackers repeatedly try leaked or common passwords against customer or admin accounts.

  • Feature: Login
  • STRIDE: Spoofing
  • ASVS: V2 Authentication / Credential Security
  • Impact: stolen accounts, order history exposure, admin compromise
  • Mitigation: password hashing, failed-attempt tracking, account lockout after 5 failures, audit logging
Demo Login
2. Payment Fraud

Checkout Price / Quantity Tampering

Attackers manipulate request values to reduce the payable amount or buy more stock than allowed.

  • Feature: Cart and Checkout
  • STRIDE: Tampering
  • ASVS: V5 Validation / Business Logic Security
  • Impact: revenue loss, fraudulent orders, stock integrity issues
  • Mitigation: server recalculates total from database, rejects client-total mismatches, logs tampering attempts
Demo Checkout
3. Admin Compromise

Unauthorised Admin Access / Privilege Escalation

Normal users or unauthenticated attackers attempt to access admin CRUD functions.

  • Feature: Admin Dashboard and Book CRUD
  • STRIDE: Elevation of Privilege
  • ASVS: V4 Access Control
  • Impact: product manipulation, order exposure, business disruption
  • Mitigation: admin-only route decorator, role checks, 403 blocking, audit logs for blocked access and CRUD actions
